Conversor de horarios

Este conversor de horarios te permitirá identificar que huso horario tiene cada país, hacer una comparativa de horarios entre países y obtener una lista de feriados/días no laborables por país. También brinda la oportunidad de, en caso de crearse una cuenta, el usuario podrá guardar información para poder acceder a ella siempre que se la necesite sin necesidad de usar el conversor nuevamente (solo requiere ingresar al perfil).
